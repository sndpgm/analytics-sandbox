class sandbox.datamodel.base.BaseData(data: StructuralDataType)[source]#

Base data class.

Parameters

data (StructuralDataType) – Input data. Supported format is pandas.DataFrame, pandas.Series, pandas.Index, numpy.ndarray, dask.dataframe.DataFrame, dask.dataframe.Series, dask.dataframe.Index, dask.array.Array.

Warning

In case of Pandas and NumPy format, values returns the actual data. However, the format of Dask returns before-compute objects, and if you want to get the actual data, you need to compute.

class sandbox.datamodel.base.BaseModelDataset(X, y)[source]#

Base class for data model of algorithm.

Parameters

  • X (StructuralDataType) – Training data. In classification model, it is for classifying and clustering the data. In regression model, it is feature vectors or matrix, but can be ignored when the regression components are not defined in the case of time series analysis.

  • y (StructuralDataType) – Target values. If algorithm is unsupervised, this should be ignored.
