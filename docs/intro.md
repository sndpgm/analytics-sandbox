# Welcome to my Analytics Sandbox

## はじめに

本サイトではアナリティクスに関する自学自習内容をログとし, 関連したコードのAPIドキュメントを記載することを目的とした**自己満サイト**です.
従って, 内容の正確性については保証しないので, 参考にする際には注意してください.
